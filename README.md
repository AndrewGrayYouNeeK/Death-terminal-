# Death Terminal — The Ultimate CLI

Brutalist terminal emulator with AI-powered autocomplete and cross-platform SSH tunneling.

## Core Concept

Death Terminal is designed as a minimal, high-contrast, no-nonsense interface that prioritizes function, speed, and control over visual polish.

## Key Features

1. **AI-powered autocomplete**
   - Context-aware suggestions for commands, command chains, and scripts based on shell history and working context.
2. **Cross-platform SSH tunneling**
   - Native SSH and tunneling workflows for Windows, macOS, and Linux, including forwarding and remote workflow support.
3. **Ultimate CLI positioning**
   - An opinionated terminal environment that integrates advanced capabilities directly into the terminal experience.

## Technology Stack

- **Zig**: performance-focused systems core.
- **Vulkan**: low-level GPU rendering for smooth terminal output.
- **gRPC**: efficient service communication for features like AI suggestions and remote workflows.
- **Lua**: lightweight scripting for customization and automation.

## Design Philosophy

- **Performance obsession**: low overhead, high responsiveness, and efficient rendering.
- **Developer-centric workflows**: AI assistance, deep customization, and built-in remote access.
- **Security-aware SSH handling**: emphasizes careful handling of credentials and tunneled traffic.
